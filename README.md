# Frontend BoilerPlate



## Run Locally

Clone the project

```bash
  git clone git@github.com:jimbertools/frontend-boilerplate.git
```

Go to the project directory

```bash
  cd frontend-boilerplate
```

Install dependencies

```bash
  yarn Install
```

Start the dev server

```bash
  yarn run dev
```
